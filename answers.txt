Q1: Why might we consider using Git?

Answer: We can use git to enable version control of our code as well as seamless collaboration between peers
Q2: What field could data science be applied to? Give an Example.

Answer: Data science could be applied to the field of finance to derive insights regarding the performance of portfolios

Q3: What is the difference between Merging and Rebasing?

Answer: Merging pushes all changes from a branch onto another branch by creating a new merge commit, while rebasing allows you to walk through each of the commits and choose which commits we want to include
